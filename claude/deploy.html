                                                          # 使用render搭建

                                                          我看貌似没人讲这个，以前貌似很多人用来着

                                                          1. 前往render进行注册open in new window
                                                          2. 确保你的GitHub账号fork了酒馆open in new window
                                                          3. 进入render，点右上角new，选择web service,选择第一个（从GitHub导入）
                                                          4. 进行GitHub授权，然后选择你fork酒馆项目
                                                          5. 配置项目名字，部署的地区，然后点击create service
                                                          6. 等待，完事

                                                          休眠

                                                          免费项目若15分钟没有访问则会自动休眠
